# melvin design system

## typography

### fonts

* primary display: signpainter
* primary text: roboto

### rules

* use lowercase text only
* do not use monospace fonts
* avoid excessive font weight variation
* keep hierarchy minimal and clear
* absolutely no emojis in the UI

## color palette

| role    | light hex | dark hex |
| ------- | --------- | -------- |
| primary | #98aa63   | #98aa63   |
| background | #ffffff | #1a1e22 |
| surface | #f4f4f4   | #2a2f36   |
| text    | #1a1e22   | #ffffff   |
| outline | #e0e0e0   | #3a4049   |
| muted   | #757575   | #6b7480   |

### usage

* primary: accents, highlights, key actions
* dark: backgrounds, primary text
* light: backgrounds, contrast text

## visual style

### principles

* minimal
* flat
* clean edges

### rules

* no shadows
* no glowing effects
* no gradients
* no rounded corners
* use subtle outlines for separation when needed

## layout

* maintain generous spacing
* align elements consistently
* avoid visual clutter
* use contrast (light vs dark) for hierarchy instead of decoration

## components

### buttons

* flat color fill (primary or dark)
* sharp corners
* no elevation or shadow
* subtle outline optional

### cards / containers

* flat surfaces only
* no rounding
* optional thin outline for separation

* simple
* direct
* modern
* always lowercase

* minimal color usage
* strict typography rules
* no decorative effects
